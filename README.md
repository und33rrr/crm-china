# CRM China - Бизнес из Китая

Веб-приложение для учёта заказов и расчёта прибыли.

## Как использовать

1. Откройте файл `index.html` в браузере
2. Все данные сохраняются локально в браузере

## Функции

### 🏠 Главная (Дашборд)
- Прибыль за сегодня / месяц / год
- Баланс Alipay
- Быстрый доступ к калькулятору и заказам

### 🧮 Калькулятор цен
- Быстро рассчитать стоимость для клиента
- Авто-доставка: вес × 6$ × 7 юань/$
- Авиа-доставка: вес × 120 юаней
- Конвертация в BYN и RUB

### 📋 Заказы
- Добавление и редактирование заказов
- Поиск по TG юзеру или трек-номеру
- Фильтрация по статусу и способу доставки
- Изменение статусов:
  - Ожидает предоплату
  - Предоплата получена
  - В пути
  - Получен мной
  - Отправлен клиенту
  - Завершён

### ⚙️ Настройки
- Курсы валют (юань/BYN, юань/RUB, доллар/юань)
- Тарифы доставки
- Баланс Alipay
- Экспорт данных в CSV

## Расчёт цен

### Авто-доставка
```
Стоимость доставки = вес (кг) × 6$ × 7 юань/$
Итого для клиента = (цена_товара + доставка) × курс_юаня + наценка
```

### Авиа-доставка
```
Стоимость доставки = вес (кг) × 120 юаней
Итого для клиента = (цена_товара + доставка) × курс_юаня + наценка
```

### Прибыль
```
Прибыль = сумма_на_Alipay - цена_товара - доставка
```

## Экспорт данных

1. Перейдите в Настройки
2. Нажмите "Экспорт заказов в CSV"
3. Файл скачается с данными всех заказов

## Советы

- Регулярно обновляйте курсы валют в настройках
- Указывайте сумму Alipay при завершении заказа для расчёта прибыли
- Используйте комментарии для важной информации
- Экспортируйте данные для резервного копирования
